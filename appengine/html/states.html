States

Many interface elements appear in diffrent states. ViUR Ignite CSS tries to consolidate recurrent state types, like .is-active, .is-disabled, .is-loading, .has-icon etc.

Some of these states have native pseudo class or attribute counterparts — :active, :disabled — or WAI-ARIA states and properties — [aria-busy=true], [aria-disabled=true].

These attributes are supported as well and in most cases recommended.
